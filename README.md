# Reservation App (MVP)

Next.js + TypeScript で作成した、予約管理のミニアプリです。  
予約の追加/一覧/削除に加えて、**時間帯の重複チェック**を実装しています。

---

## Features

- 予約の追加（予約者名 / 開始日時 / 終了日時 / メモ）
- 予約一覧の表示
- 予約の削除
- **予約の重複チェック**（既存予約と時間帯が重なる場合は登録不可）

---

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
